Parameters are the frozen canonical V27 set from usdt-slope-research, verified trade-for-trade and not re-fitted — so the evaluation window runs to the latest settled bar and is refreshed daily. Costs included: 5 bps/side fee+slippage and 3 bps/day long funding on 3x notional; close-based isolated-margin liquidation modeled (none occurred). BTC buy&hold over the same window: +46.2%.
